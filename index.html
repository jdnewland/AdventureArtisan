Adventure Artisan

About

Adventure Artisan is a web tool designed to help Dungeon Masters quickly generate engaging D&D adventure prompts. Select a prompt style from the dropdown menu and click "Generate Adventure" to create a new scenario.

Prompt Styles

  • Quest for Glory: Simple adventure prompts.
  • Wheel of Destiny: Complex prompts involving multiple elements.
  • Battle of Minds: Conflicting goals leading to the same obstacle.
  • Fractured Soul: Internal conflict with a forced choice.
